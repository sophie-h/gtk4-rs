Gets the contents of self stored in value.

The value will have been initialized to the GType the value should be provided in. This given GType does not need to be listed in the formats returned by ContentProviderExt::formats(). However, if the given GType is not supported, this operation can fail and gio::IOErrorEnum::NotSupported will be reported.
